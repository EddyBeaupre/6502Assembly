LDA - 6502 LDA, Load Accumulator

Information about the 6502 LDA Instruction
LDA --- LDA stands for LoaD Accumulator
LDA is one of the Memory Operations in 6502 instruction-set. The function of LDA operator is to load a byte of memory into the accumulator and setting the zero and negative flags as appropriate. Click here to see the opcode of LDA Instructure.
Status flag of 6502 LDA instruction
Only Zero Flag and Negative Flag are affect. For Detail, please see status flag summary of LDA.
Addressing Mode Summary for 6502 LDA instruction

All the addressing modes are suitable for the LDA instruction, for detail, see the Addressing Mode Summary of LDA instruction:

  1. Immediate,
  2. Zero Page
  3. Zero Page, X
  4. Absolute
  5. Absolute, X
  6. Absolute, Y
  7. (Indirect X)
  8. (Indirect) Y
