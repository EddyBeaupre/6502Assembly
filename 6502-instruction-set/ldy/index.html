LDY - 6502 LDY, Load Register Y

Information about the 6502 LDY Instruction
 
LDY    ---   LDY stands for  Load Register Y
LDY is one of  the Memory Operations in 6502 instruction-set. The function of LDY instruction is to load a byte of memory into the  Index Register Y and to set  the zero and negative flags as appropriate

Possible flag status of 6502 LDY Instruction

7 6 5 4 3 2 1 0
Negative Overflow   Break Decimal Interrupt Zero Carry

The Addressing Mode Summary of 6502 LDY Instruction
 
Addressing Mode Summary
Addressing Mode Example Explanation Example Opcode Bytes Cycles
Immediate LDY #10 Load $10 (Decimal 16) into Index Register Y A0 10 2
Zero Page LDY $00 Load Index Register Y from zero page address $00 A4 00 3
Zero Page,X LDY $10,X Load Index Register Y from zero page with address calculated from $10 adding content of Index Register X B4 10 4
Absolute LDY $1234 Load Index Register Y from address $1234 AC 34 12 4
Absolute,X LDY $1234,X Load Index Register Y from address calculated from $1234 adding content of Index Register X BC 34 12 4
(+1 if page crossed)
