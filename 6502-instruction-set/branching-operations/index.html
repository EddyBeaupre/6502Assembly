Branching Operations

Branching operations:

  1. BCC    ---    Branch on Carry Clear
  2. BCS    ---    Branch on Carry Set
  3. BEQ    ---    Branch on EQual (zero set)
  4. BMI    ---    Branch on MInus (negative set)
  5. BNE    ---    Branch on Not Equal (zero clear)
  6. BPL    ---    Branch on PLus (negative clear)
