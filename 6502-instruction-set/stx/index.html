STX - 6502 STX, Store Register X

Information about the 6502 STX Instruction

 STX --- STX stands for STore Register X

STX is one of the Memory Operations in 6502 instruction-set. Stores the contents of the Index Register X to specific memory address and set the flags accordingly.
