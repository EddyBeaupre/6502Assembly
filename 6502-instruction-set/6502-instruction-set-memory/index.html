6502 Memory Operations

Memory Operations of 6502 Instruction sets

Most of the operations in microprocessors involve moving data between memory and registers. Memory Operations are important in the fundamental operations of 6502 processors. Memory operations can be categories into two main categories: Load Operations and Store Operations.

Load Operations are instructions that get the content from memory (RAM / ROM) and store the content to registers for further operations.
Store Operations are instructions that get the content from registers and store the content back to memory (RAM) after finishing some operations.

There are six Memory Operations in 6502 instruction-set:
  1. LDA --- LDA stands for Load Accumulator
  2. LDX --- LDX stands for LoaD X
  3. LDY --- LDY stands for LoaD Y
  4. STA --- STA stands for STore Accumulator
  5. STX --- STX stands for STore X
  6. STY --- STY stands for STore Y

  • Note: STZ (STore Zero) is an operator in 65C02 Instruction-set.
