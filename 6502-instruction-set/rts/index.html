RTS - 6502 RTS, Return from Subroutine

Information about the 6502 RTS Instruction

RTS --- RTS stands for Return from Subroutine

RTS is one of the 6502 Subroutine Operations of 6502 instruction-set. The function of RTS is to pulls the top two bytes off the stack (low byte first) and transfers program control to that address+1 i.e.return (RTS) from the calling subroutine (6502 JSR instruction).
