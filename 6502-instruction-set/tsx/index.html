TSX - 6502 TSX, Transfer Stack pointer to X

Information about the 6502 TSX Instruction

TSX --- TSX stands for Transfer Stack pointer to X

TSX (Transfer Stack pointer to X) is one of the Register transfer operations in 6502 instruction-set. TSX operation transfers the content of the Stack Pointer to Index Register X and sets the zero and negative flags as appriopriate
