Registers

The 6502 family microprocessors usually have six registers:
    1. one 8-bit accumulator register (A)
    2. two 8-bit index registers (X and Y)
    3. an 8-bit processor status register (SR)
    4. an 8-bit stack pointer (SP)
    5. and a 16-bit program counter (PC)
