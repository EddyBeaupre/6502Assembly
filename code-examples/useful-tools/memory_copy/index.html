Memory_Copy 

;---------------------------------------------
;Memory_Copy:
; Purposes:
;	1. Copy a predined block of memory to specific address.
; Input: 
;	1. Source: tmp3:tmp2
;	2. Destination: tmp5:tmp4
;	3. Length: X
; Output:
; Required Variables:
; Required Functions:
; Required Table:
; Labels used:
; Variables used:
; Constant used:
; Function used:
; Used by:
; Version: 1.0
; Revision Date: 28/8/2006
;---------------------------------------------
Memory_Copy:
	LDA (tmp2)
	STA (tmp4)
	INC tmp2
	BNE Memory_Copy_01
	INC tmp3
Memory_Copy_01:
	INC tmp4
	BNE Memory_Copy_02
	INC tmp5
Memory_Copy_02:
	DEX
	BNE Memory_Copy 
Memory_Copy_Exit:
	RTS
;End of Memory_Copy
;---------------------------------------------
